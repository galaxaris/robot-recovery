# \## Features

# 

# The \_Omicronde Project\_ is platformer video game project. Intended for children between 6 and 10 years old, it is a fun and educational game about our future and the importance of taking care of our planet... 

# 

# You will play a little robot responsible for cleaning up the planet and fighting pollution. You will explore different environments, solve mysteries, and fight bosses to save the world!

# 

# \*\*Still in development (Beta)! This is, for now, a demo of the Game Engine (based on the \_Omicronde API\_).\*\*
